GovData

Jsou vysoce dostupnou službou poskytující vybraná open data k přímému použití ve webových a mobilních aplikacích.

Důvody vzniku služby shrnujeme v článku Proč vznikají GovData?

Fakta

Proč používat GovData?

  • Budujeme otevřená SDK pro nejpopulárnější jazyky
  • API je vysoce dostupné a škálovatelné
  • API má vysoké limity a krátké odezvy
  • SSL zabezpečení s platnými certifikáty je standard

Státní online služby mají tyto nedostatky:

  • Nejsou vysoce dostupné a škálovatelné
  • API mají nízké limity a velmi dlouhé odezvy
  • API nemají SDK
  • Nemají platné SSL certifikáty
  • Nemají spolehlivé servery a díky nim velmi dlouhé odezvy požadavků
  • Výstupy API pouze v XML v době masivní adopce JSONu
  • Nemají kvalitní dokumentaci
  • Mají nestandardní podmínky použití

Dostupná data
